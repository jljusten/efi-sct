Welcome to the EFI Self-Certification Tests (EFI-SCT) Project on the EFI and Framework Open Source Community Website.

The EFI-SCT is no longer maintained on this web site and the UEFI SCT has been moved to the UEFI Forum at html://www.uefi.org members page.

The UEFI SCT requires the EFI Developer Kit (EDK) and the EFI Shell to compile and build. Please refer the EDK Getting Started Guide (pdf) to install the EDK and the EFI Shell Getting Started Guide (pdf) to install the EFI Shell.
